# PARADOX

The digital companion for the Paradox effect by Alex Latorre & Judah Gabriel.

A minimal, dark-mode experience designed to teach the Paradox magic trick.

## The Stack

- **Core**: Vanilla web technologies.
- **Style**: Deep blacks (`#0a0a0a`), electric accents, and modern typography.
- **Performance**: Zero dependencies. Instant load times.
